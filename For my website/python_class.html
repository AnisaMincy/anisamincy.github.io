About This Class

This class will teach you the basics of Python programmming and you'll create three python projects.

The only materials you will need are an Repl.it account and access to google classroom, and zoom. The class is ten days long with one hour long zoom meetings each day. Every day you will learn new concepts, practice those concepts and take a quiz. This course is designed for all ages.
