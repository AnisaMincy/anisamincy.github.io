Game Design With Scratch

Scratch is a super fun and easy way to learn coding and an awesome way to make games!

About This Class

This class will teach you the basics of programming, learn how to use scratch, and create fun games!

The only materials you will need are an Scratch account, a google account, and access to Zoom. The class is eight days long with one hour long zoom meetings each day. Every day there will be a new game to make in scratch and an instructional lesson. This course is designed for ages 7-15.

If you are having any trouble feel free to contact me!
